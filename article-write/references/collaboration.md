# 人机协作边界

本 skill 是写作助手，不替代作者判断。

## AI 适合做

- 找论据、案例、数据、历史类比（须可核实或标来源）
- 提供比喻候选
- 在**已定角度与分段标题**下扩写
- 梳理结构、转场建议
- 技术背景、步骤表述（教程须可复现或标「待验证」）

## 必须人来

- 第一手观察与真实经历
- 核心创意角度（「对，就是这个」）
- 情绪的真实体感（「愣住了」≠「很震撼」）
- 从数据到具体人物的共情（须真人感受）

## 推荐协作顺序

```text
人：素材 + 观点 + 经历 + 情绪节点
  → AI：背景 + 证据 + 结构建议 + 按角度扩写
  → 人：二次改写（声音、节奏、真实细节）
  → AI：qa-layers 检查 + 修改建议
  → 人：终审
```

无用户经历时：用提问占位，或写「我尚未验证，但好奇…」，禁止编造「比如有一次…」。
