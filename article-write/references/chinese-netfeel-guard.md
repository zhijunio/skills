# 中文网感守护（小红书 / 公众号 / 脚本）

当内容「像外国人假装写中文」、中途变英文、或情绪化表达被稀释时使用。

## 核心问题

- 句式僵硬、文化框架西化（In conclusion、线性问题-解决方案）
- 长文后半段网感崩塌
- 模型在复杂生成中回退英文思维链

## 工作流

### 1. 强制中文思维链（生成前）

用中文写出：选题最打动自己的时刻、读者痛点、情绪曲线、会用的口头禅。**不要**英文思考再翻译。

### 2. 泄露检测（生成后）

扫描：生硬「姐妹们谁懂啊」模板句、Furthermore/In conclusion、美剧式叙事、句长过于均匀、缺少第一人称翻车细节。

输出：泄露片段 + 严重程度。

### 3. 网感重写

针对问题段：口语 + 情绪 + 短句 + 停顿；参考 `chinese-netfeel/真实高赞网感示例.md`；平台语气见 `chinese-netfeel/平台网感适配器.md`。

### 4. 多版本 + 自检

输出 2–3 个情绪强度版本；网感评分 1–10；Before → After 对照。

## 安全契约

- 只改语言与文化表达，不改变核心观点、结构、数据
- 尊重用户 CLAUDE.md / AGENTS.md 人设
- 原稿网感已够好时只做微调建议
