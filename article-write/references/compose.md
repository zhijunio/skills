# Compose（成稿阶段）

从素材产出正文。先读 `intake.md`、`collaboration.md`，再选 **form**。

## 1. Intake

- HKR 不及格 → 先 `ideate` 或与用户对齐，不硬写
- 缺经历/观点 → 提问或 `[待作者补充]`

## 2. 选形态（form）

| form | 文件 |
|------|------|
| long | `long-form.md` — 原型、节奏、标题、结构 |
| tutorial | `tutorial-form.md` — 说人话步骤、风险 |
| script | `script-form.md` |
| speech | `speech-form.md` |
| xiaohongshu | `xiaohongshu-form.md` |

## 3. 声音

全程执行 `voice-default.md`（或用户指定声线）。

## 4. 深度

| depth | 动作 |
|-------|------|
| outline | 只交标题 + 大纲 + 每段要点 |
| draft | 全文；完稿后过 L1 |
| publish-ready | 全文 + 完整 `qa-layers.md` 报告 |
