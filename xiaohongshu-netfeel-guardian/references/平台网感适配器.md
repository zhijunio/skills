# 平台网感适配器（v1.0）

## 小红书种草笔记
- 核心语气：姐妹感 + 强烈情绪共鸣 + 场景细节 + “谁懂啊”式真实吐槽
- 句式特征：短句、口语化、带情绪起伏、常用“绝绝子”“泪目”“yyds”（适度）
- 禁忌：过于正式、鸡汤翻译腔、西方励志框架

## 公众号深度文章
- 核心语气：有故事感 + 情绪曲线 + 真实个人经历 + 稍正式但不失人味
- 句式特征：自然过渡、情绪递进、避免“In conclusion”
- 禁忌：纯翻译体、过度线性逻辑

## 视频脚本（口播）
- 核心语气：像真人聊天 + 停顿感 + 情绪感染力 + 适合朗读
- 句式特征：短、重复强调、口语化、带“大家好，我是XX”
- 禁忌：书面长句、翻译腔

使用时请根据目标平台选择对应适配器。
