# Learning record format

Records live in **`learning-records/NNNN-slug.md`**, numbered sequentially (adapted from [mattpocock/skills teach LEARNING-RECORD-FORMAT](https://github.com/mattpocock/skills), MIT).

## Template

```markdown
# {Short title}

{1–3 sentences: what was learned or corrected, and why it matters for future runs.}
```

Optional when useful:

- **Status:** `active` | `superseded by LR-NNNN`
- **Evidence:** demo output, source read, user stated prior knowledge
- **Implications:** what to teach or skip next time

## When to write

1. **Non-obvious misconception corrected**
2. **Prior knowledge depth established** («我已经会 X» — record depth claimed)
3. **Mission or boundary changed** — cross-update mastery-map Mission

## When not to write

- Material merely covered (coverage ≠ learning)
- Duplicate glossary definitions
- Session activity logs

## Supersession

If understanding replaces an earlier record, mark old `superseded by LR-NNNN`; do not delete.
