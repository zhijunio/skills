# Vet

Before delivery, check for over-editing and missed tells.

## Downgrade or revert when

| Class | Action |
|-------|--------|
| Meaning shifted | Revert phrase; tells fix only |
| Technical precision lost | Restore term; fix surrounding filler only |
| User forbidden list hit | Revert |
| Removed all hedging on uncertain claims | Restore one honest modal |
| Added personality facts not in draft | Delete addition |
| Structure changed | Revert headings/order |

## Stop conditions

- touch: no further passes after one detect+rewrite
- standard: max two rewrite passes + one vet
- deep: L1–L4 checklist all pass or documented exception

## False positive tells

Do not "fix":

- Domain-standard jargon the author uses consistently
- Quoted speech or cited text
- Intentional list format (API reference, changelog)
- Deliberate formal register (legal, academic) unless user asked to casualize
