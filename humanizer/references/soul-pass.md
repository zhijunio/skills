# Soul Pass

Run after tell-removal. Goal: **human texture**, not new content.

## Do

- Vary sentence length within a paragraph
- Allow one direct opinion **if the draft already implies stance** ("I don't buy this" when draft was skeptical)
- Replace vague intensity with **existing** specifics from the same draft
- Leave one slightly imperfect transition — not every paragraph needs a thesis sentence

## Don't

- Invent interviews, statistics, or "a friend of mine"
- Add humor the draft didn't invite
- Turn technical docs into memoir
- Replace all hedging with bravado — accuracy beats voice

## Sterile after scrubbing?

If text reads like neutral encyclopedia:

1. Pick the author's strongest concrete detail and move it earlier in the paragraph
2. Break one long sentence into spoken rhythm
3. Delete one redundant summary sentence at paragraph end

Stop when a read-aloud sounds like a person explaining, not a template.
