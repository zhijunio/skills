# QA Layers (deep mode)

Run after revised text exists. Fail any layer → fix and re-check.

## L1 — Content integrity

- [ ] Numbers, names, dates unchanged (or flagged to user)
- [ ] Links and code intact
- [ ] No new claims or citations
- [ ] Heading structure unchanged

## L2 — Tell removal

- [ ] No class 19–21 chatbot artifacts
- [ ] No significance inflation (class 1) in opening/closing
- [ ] No rule-of-three stacks (class 10)
- [ ] Lane file spot-check: top 10 phrase hits cleared

## L3 — Voice

- [ ] Not sterile Wikipedia — at least one human rhythm cue per ~300 字 / ~200 words
- [ ] Register matches intake (technical/casual/formal)
- [ ] No fabricated anecdotes

## L4 — Read-aloud

- [ ] Full piece read aloud mentally — no stumble on connector chains
- [ ] Opening hooks with substance, not throat-clearing
- [ ] Ending doesn't moralize or sloganize

**Pass bar:** L4 feels like a person wrote it; any "obviously AI" line triggers rework.
