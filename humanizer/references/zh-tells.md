# Chinese Tells

Apply when `language_lane: zh`. Cross-check `patterns.md` classes.

## High-signal phrases (delete or replace)

| Tell | Prefer |
|------|--------|
| 值得注意的是 / 不难发现 / 综上所述 | Delete or one plain connector |
| 首先…其次…最后… / 一方面…另一方面… | Merge or pick one thread |
| 在当今…时代 / 随着…的不断发展 | Delete opener; start with subject |
| 作为…的重要组成部分 / 扮演着关键角色 | X 用于 Y / X 负责 Y |
| 赋能 / 深耕 / 抓手 / 闭环 / 落地 / 助力 | Plain verb: 用、做、改、发布 |
| 极具 / 深刻 / 令人瞩目 / 毋庸置疑 | Delete or cite one metric |
| 不仅…而且… / 既…又…（连用） | One claim |
| 从某种意义上说 / 在一定程度上 | Delete or be specific |
| 相关专家表示 / 业内人士指出 | 谁 + 在哪说的 |
| 让我们… / 不妨… / 接下来我们将 | Delete meta; state content |
| 希望对你有所帮助 / 如有疑问欢迎 | Delete |

## Structure tells

- **金句结尾** — sounds poster-ready → rewrite as plain statement
- **排比三连** — keep two or break rhythm
- **过度书名号/引号强调** — reduce decorative quoting
- **段段以总结句收** — vary endings; some paragraphs can stop mid-thought
- **列表项全是「XX：……」** — merge into prose or shorten bullets

## Rhythm

- Mix 5–15 字短句 with longer explanation sentences
- Technical posts: allow one术语 per sentence; don't stack four abstractions
- Allow口语连接: 但、所以、其实、说白了 — sparingly, not every paragraph

## Preserve unless user says otherwise

- 你 vs 您 (match draft register)
- Established product names and English identifiers
- Deliberate colloquialism the author already uses
